• subtitlesKindle Pocket 使用说明
    Kindle Pocket拥有几千本电子书资源，当您需要电子图书的时候，向Kindle Pocket发出您的请求， Kindle Pocket就会将您需要的书籍发送到您绑定的指定邮箱。

    Kindle Pocket悄悄告诉您，如果您使用的设备是kindle，那会更加便捷， Kindle Pocket会将电子书籍发送到您绑定的Kindle邮箱，由亚马逊服务器负责推送到您的Kindle设备中；

    当然了，Kindle Pocket只为您服务，是不作商业用途的。
  • subtitlesKindle Pocket 是如何为您服务的
    【搜索】在Kindle Pocket的对话界面，输入您想获取的书籍名称或关键字。例如：输入关键字“红楼梦”， Kindle Pocket就会为您显示所有带有“红楼梦”关键字的书籍

    【发送请求】点击进入具体书目页面，发送请求。例如：您想阅读俞平伯的《红楼梦辩（中华现代学术名著丛书）》， 点击“send”按钮，这样，Kindle Pocket就能轻松地获取您的需求，并以最快的速度将书籍发送到您绑定的指定邮箱中。 看见“comment”键了吗？点击它，您就可以尽情表达对这本书的看法。


    【下载】进入绑定的指定邮箱查看并下载。或者在Kindle中直接查看推送的书籍。

    【个性“+”】在进入搜索结果页面后，点击右下角“+”键，可以查看Kindle Pocket为您设置三个个性功能键：个人信息，意见和建议，以及指南。 在个人信息页面中，您可以查看账户绑定信息并更新，查看推送记录、评论记录和意见记录。

    【注册及邮箱的绑定】在Kindle Pocket对话界面，输入数字“2”或“绑定”、“更新”，就可以进入注册和邮箱绑定的界面（具体的填写方法Kindle Pocket会在【绑定步骤】里详细说明的）。

    【返回本页面】在Kindle Pocket对话界面，输入数字“1”或“介绍”、“指南”，就可以重新接收进入本页面的链接。
  • subtitles如何绑定邮箱
    【步骤】

    我们在Kindle Pocket对话界面输入“2”或“绑定”、“更新”，进入以下界面。 您可以按照图中要求依次填写，最后一项“kindle邮箱”里的内容就是您绑定的指定邮箱，也就是接收电子图书的邮箱（推荐填写Kindle邮箱）。

    【注意事项】

    1. 若您使用启用了独立密码的QQ邮箱，请使用独立密码进行绑定（或授权码）而非QQ邮箱登录密码；

    2. 个人邮箱暂只支持163,126和QQ邮箱，其他邮箱的推送功能也将在今后逐步完善；

    3. 本公众号电子书资源以mobi格式文件为主，在除Kindle设备外的其他电子书阅读软件上可能无法浏览；另有pdf，txt，epub等其他格式的资源文件；

    4. 如果您所要推送的邮箱是Kindle邮箱，请将个人邮箱添加到亚马逊Kindle设备的信任邮箱列表中；如何查看Kindle邮箱？⬇️️  如何添加信任列表？⬇️️

    5. 请确保个人邮箱已经开启了【IMAP/SMTP服务】；如何开启？⬇

    6. 为避免个人邮箱设置异常，本公众号会在您的个人邮箱授权失败后使用公共邮箱账号【kindlepocket@163.com】进行推送， 所以当您的目的邮箱为Kindle邮箱时，请务必将此公共邮箱账号添加到Kindle信任列表中，以防止在个人邮箱推送失败后，无法推送的问题； 请勿长期使用此公共邮箱账号进行推送，当您在个人信息页面的推送记录中查看到个人邮箱推送失败的信息后，务必及时按照此指南更新您的个人邮箱设置，特别是【IMAP/SMTP服务】的设置 以免本公共邮箱账号被亚马逊列入黑名单而导致您无法接收电子书。 ️
  • subtitles如何查看Kindle邮箱账号
    1. 启动Kindle设备并登录亚马逊账号；

    2. 点击 “设置 ->我的账户”；

    3. 查看 “【发送至Kindle】电子邮箱地址”；

    【提示】当您的Kindle设备登录了亚马逊账户后，您使用个人邮箱（已添加到Kindle信任邮箱列表中的）推送到Kindle设备邮箱的文件，就可以被亚马逊服务器推送到Kindle设备中。
  • subtitles如何添加Kindle信任邮箱
    1. 登录亚马逊网站；

    2. 点击 “我的账户 -> 管理我的内容和设备 -> 设置 -> 个人文档设置”；

    3. 查看 “已认可的发件人电子邮箱列表”；

    4. 点击 “添加认可的电子邮箱”；

    5. 将您需要绑定的个人邮箱添加到此邮箱列表中。
  • subtitles如何启用IMAP/SMTP服务和授权码
    1. 以QQ邮箱为例；请登录QQ邮箱；

    2. 点击 “设置 -> 账户 -> POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务”；

    3. 查看IMAP/SMTP服务，点击“开启”；可能需要您进一步验证密保；

    4. 开启后会提示您的授权码，或直接点击下方的“生成授权码”来获取；您可以在本公众号的绑定界面中，将个人邮箱的密码填写为本授权码；

    5. 其他类型邮箱（如163，126）操作类似。
